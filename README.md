## Rodando os projetos.
### Rodando com docker
Rodando ```docker build .``` no diretório onde se encontra o Dockerfile
### Rodando com DotNet
Entrando no namespace do projeto por exemplo na pasta HelloWorld e executando
```dotnet run```
